
# Simplified

### Passo a passo
Suba os containers do projeto
```sh
docker-compose up -d
```

Crie o Arquivo .env
```sh
cp .env.example .env
```

Acesse o container app
```sh
docker-compose exec app bash
```

Instale as dependÃªncias do projeto
```sh
composer install
```

```sh
php artisan key:generate
```

```sh

```

[http://localhost:8000](http://localhost:8000)
